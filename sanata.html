Behind SANATA

From using tools to building one.

SANATA represents a new part of Romanich’s practice: designing the instrument behind the music. The project is currently a working prototype, and this page documents its sound while the interface and feature set continue to develop.

No compatibility or public-release date is announced yet. Future updates can add screenshots, technical details, presets and download information when they are ready.
